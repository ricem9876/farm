🎮 READY-TO-USE CODE SNIPPETS

Add muzzle flash:
  @onready var muzzle_flash = $MuzzleFlash
  muzzle_flash.restart()

Add bullet impact:
  EffectsManager.create_simple_impact(position, Color.YELLOW)

Add enemy death:
  EffectsManager.create_simple_impact(position, Color.RED, 30)

Add sparkles:
  EffectsManager.create_sparkle(position)
