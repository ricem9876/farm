🎨 WANT TO CUSTOMIZE?

Each effect can be opened and edited in Godot:
   1. Double-click the .tscn file
   2. Select GPUParticles2D node  
   3. Adjust settings in Inspector
   4. Save (Ctrl+S)

Common adjustments:
   • Amount - more/fewer particles
   • Lifetime - how long they last
   • Initial Velocity - how fast they shoot
   • Color Ramp - change the colors
   • Scale - size of particles
