## ğŸ”« MUZZLE FLASH (For Your Gun)

### In Gun.gd (or wherever you handle shooting):

```gdscript
func shoot():
    # Your existing bullet spawn code here...
    spawn_bullet()
    
    # Add this line to create muzzle flash:
    EffectsManager.play_effect("muzzle_flash", global_position, rotation_degrees)
```

**That's it!** The muzzle flash will appear at your gun's position.

### Adjusting Direction:
The muzzle flash points in the direction of the gun's rotation.
- If your gun points right by default, it'll work automatically
- If your gun uses a different system, adjust the rotation parameter

## âœ¨ LOOT SPARKLE (On Weapon Chests/Pickups)

### Option 1: Continuous Sparkle (Recommended for chests)

In WeaponChest.gd or your pickup script:

```gdscript
func _ready():
    # Spawn sparkle effect that stays
    var sparkle = EffectsManager.spawn_oneshot_effect(
        EffectsManager.effects["loot_sparkle"],
        global_position,
        self  # Parent it to this object
    )
```

**Note:** The loot sparkle is set to emit continuously, so it will keep
sparkling until the chest is opened/picked up.

### Option 2: One-Time Sparkle Burst

For when an item is collected:

```gdscript
func _on_collected():
    # Create sparkle burst
    EffectsManager.play_effect("loot_sparkle", global_position)
    
    # Your collection code...
    add_to_inventory()
    queue_free()
```

## ğŸ¨ CUSTOMIZING THE EFFECTS

Want to change colors or behavior? Here's how:

1. **Open the effect scene** in Godot:
   - Double-click: `Resources/Effects/MuzzleFlash.tscn` (or any effect)

2. **Select the GPUParticles2D node**

3. **In the Inspector, expand "Process Material"**

4. **Adjust these settings:**
   - **Amount**: More/fewer particles
   - **Lifetime**: How long particles exist
   - **Initial Velocity**: How fast particles shoot out
   - **Spread**: How wide the cone of particles
   - **Gravity**: Makes particles fall/rise
   - **Scale**: Size of particles
   - **Color Ramp**: Click to change colors

5. **Save the scene** (Ctrl+S)

## ğŸ® EXAMPLE: Complete Gun Script

Here's a full example showing how it all fits together:

```gdscript
extends Node2D

@export var bullet_scene: PackedScene
@export var fire_rate = 0.2
var can_shoot = true

func _process(delta):
    if Input.is_action_pressed("shoot") and can_shoot:
        shoot()

func shoot():
    can_shoot = false
    
    # Spawn bullet
    var bullet = bullet_scene.instantiate()
    get_tree().current_scene.add_child(bullet)
    bullet.global_position = global_position
    bullet.rotation = rotation
    
    # MUZZLE FLASH!
    EffectsManager.play_effect("muzzle_flash", global_position, rotation_degrees)
    
    # Cooldown
    await get_tree().create_timer(fire_rate).timeout
    can_shoot = true
```
